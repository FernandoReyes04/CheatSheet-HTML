HTML Cheat Sheet   FernandoReyes F1 Team

Estructura HTML
  • <!DOCTYPE> - Define la version HTML
  • <html> - Elemento raíz contenedor
  • <head> - Contiene metadatos
  • <body> - contiene contenido visible
Contenido Semántico
  • <header> - Encabezado de la página
  • <nav> - Navegación principal
  • <main> - Contenido principal
  • <aside> -Contenido relacionado
  • <footer> - Pie de página
  • <article> - Contenido independiente
  • <section> - Agrupación lógica
Multimedia
  • <img> - Implementar imagenes
  • <audio> - Implementar audio
  • <video> - Implementar videos
  • <source> - Fuente alternativa
Texto y contenido en línea
  • <p> - Parrafo de texto
  • <span> - contenedor genérico
  • <a> - Enlace a otra página
Listas
  • <ol> - Lista ordenada numerada
  • <ul> - Lista desordenada viñetada
  • <dl> - Lista de definción
  • <li> - Elemento de lista
  • <dt> - Termino de definición
  • <dd> - Descripción de un termino
Tablas
  • <table> - Creación de tabla
  • <tr> - Fila de la tabla
  • <td> - Celda de la tabla
  • <th> - Celda de Encabezado
  • <thead> - Encabezado de la tabla
  • <tbody> - Cuerpo de la tabla
  • <tfoot> - Pie de la tabla

Elementos Interactivos
  • <form> - Formulario de entrada
  • <input> - Entrada de datos
  • <select> - Menú desplegable
  • <option> - Opción dentro del menú
  • <textarea> - Área de texto grande
  • <label> - Etiqueta para campo form
Incrustaciones externas
  • <iframe> - Incrustar otra pagina
  • <object> - Incrustar recurso externo
